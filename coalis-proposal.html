1Claude Code とは -- なぜこれが最強なのか

普段お使いのサービスすべてに、1つの画面からつながる新しい働き方。

1つのCLIからすべてに接続

Claude Code は、Anthropic社が提供する最新のAIインターフェースです。最大の特徴は、Gmail、Googleカレンダー、Googleスプレッドシート、Googleドライブ、Slack、Web検索など、普段お使いのサービスすべてに「たった1つの画面」から接続できることです。

従来であれば、メールを確認するためにGmailを開き、予定を見るためにカレンダーを開き、チャットを確認するためにSlackを開き......と、1日に何十回も画面を切り替えていたはずです。Claude Code を導入すると、この「画面を切り替える」という作業そのものがなくなります。すべてが1つの場所で完結します。

自然な言葉で操作できる

Claude Code への指示は、日本語の自然な言葉で行います。コマンドや専門用語を覚える必要はありません。たとえば：

  • 「今日の予定教えて」と入力するだけで、Googleカレンダーから本日のスケジュールが表示されます
  • 「未読メールある？」と聞くだけで、Gmailの未読メールが要約付きで一覧されます
  • 「この業界の最新ニュース調べて」と頼むだけで、Web検索を行い結果をまとめてくれます

AIが会話の文脈を理解する

Claude Code は、会話の流れ（文脈）を理解します。たとえば、最初に「〇〇業界について調べて」と依頼し、その調査結果を見た後に「さっき調べた内容をスプレッドシートにまとめて」と続けるだけで、AIは「さっきの調査結果」が何を指すかを理解して作業してくれます。

これは単なる検索エンジンやチャットボットとは根本的に異なるポイントです。AIがあなたの仕事の流れに沿って、連続した作業を進めてくれるのです。

接続するサービス一覧（MCP）

MCPとは「Model Context Protocol」の略で、Claude Code と各サービスをつなぐ接続の仕組みです。以下のサービスに接続し、それぞれの機能をClaude Code から直接操作できるようになります。

サービス できること 具体例
Gmail メールの確認、検索、下書き作成、送信補助 「未読メールを重要度順にまとめて」「〇〇さんに返信のドラフト作って」「今週のメールで投資案件に関するものだけ抽出して」
Google Calendar 予定の確認、新規作成、変更、空き時間の検索 「今週の予定を一覧にして」「明日14時にMTGを入れて」「来週の空き時間を教えて」
Google Sheets データの読み書き、集計、分析、グラフ作成 「ポートフォリオの月次データを集計して」「前年比の推移をグラフにして」「KPIシートを更新して」
Google Drive ファイルの検索、閲覧、アップロード、整理 「先月のDD資料どこにある？」「この資料をドライブに保存して」「投資委員会フォルダの中身を見せて」
Slack メッセージの送受信、チャンネル検索、スレッド確認 「〇〇チャンネルの未読まとめて」「チームに進捗報告して」「昨日の議論の要点を教えて」
Web検索 ニュース検索、業界リサーチ、企業情報収集 「この業界のM&A動向を調べて」「〇〇社の最新ニュースは？」「日経平均の今日の動きは？」
ポイント: これらのサービスは「接続」するだけで、データがどこかに送られるわけではありません。APIキー（接続用の鍵）はすべてお手元のパソコン内に保存されます。セキュリティについては後述の「セキュリティ・安全性」セクションで詳しくご説明します。

2AI Crew とは -- 6人のプロフェッショナルチーム

投資ファームの精鋭チーム「The Firm」。6人のプロフェッショナルが、上原さんの業務をあらゆる面からサポートします。

世界観: The Firm

AI Crew は、1つのAI（Claude Code）に「6つの人格（キャラクター）」を設定したものです。キャラクターごとに専門分野・口調・対応範囲が異なり、名前で呼びかけるだけで切り替わります。

Coalis様向けの世界観は「The Firm（ファーム）」。PE/FA業務を支えるプロフェッショナルチームというコンセプトで、落ち着いた知性と機動力を兼ね備えた頼れるパートナー集団です。各キャラクターはそれぞれ異なる強みを持ち、チームとして連携して業務を支えます。

使い方のイメージ: 「まどか、今日の予定教えて」「恭介、この業界調べて」のように、名前で呼びかけるだけでキャラクターが切り替わります。特に指定しなければ、総合窓口のまどかが対応し、必要に応じて他のメンバーに引き継ぎます。

3導入するファイル一覧

Claude Code に設定するファイルの全体像です。それぞれのファイルが何をするのか、なぜ必要なのかをご説明します。

Claude Code は、特定のフォルダにテキストファイルを置くことで動作をカスタマイズできます。プログラミングの知識は不要です。テキストファイルに「こういう風に振る舞ってほしい」と日本語で書いておくだけで、AIがその指示に従います。

ファイル名 保存場所 役割・説明
CLAUDE.md ~/.claude/CLAUDE.md メイン設定ファイル（最も重要）
Claude Codeの「性格」と「ルール」を定義するファイルです。上原さんの情報（名前・会社・業務内容）、安全ルール（削除禁止・確認必須など）、使用するサービスの一覧、キャラクターの概要がすべてここに書かれています。Claude Codeを起動するたびに、まずこのファイルを読み込んでから対話を開始します。
characters.md ~/.claude/characters.md 全キャラクターの詳細定義
6人のキャラクター全員の口調、できること、使うツール、制約事項がまとめて定義されています。CLAUDE.mdから参照されており、キャラクターの振る舞いを一括管理できます。
まどか.md ~/.claude/agents/ まどか個別の詳細設定

まなみ.md ~/.claude/agents/ まなみ個別の詳細設定
データ分析の手法、スプレッドシートの操作ルール、グラフのスタイル設定、数値の取り扱い注意点が定義されています。
くるみ.md ~/.claude/agents/ くるみ個別の詳細設定
文書作成のテンプレート、投資メモの構成ルール、LP向けレポートのフォーマット、文体のガイドラインが定義されています。

Gmail・カレンダーの操作ルール、メール下書きのフォーマット、スケジュール調整時の確認フロー、リマインドのタイミングが定義されています。

ニュース収集の情報源、X投稿のトーン・文字数ルール、煽り表現禁止ルール、ニュースレターのフォーマットが定義されています。

「おはよう」と話しかけるだけで毎朝の定型チェック（予定確認・メール確認・ニュース収集・マーケット概況）を自動実行するための設定ファイル一式です。

Scene 02

Gmailを開かずに、メールの確認・検索・返信ドラフト作成がすべてClaude Codeから行えます。特に、大量のメールから特定の案件に関するものだけを抽出したり、返信の文面を自動生成する機能は、日々の業務時間を大幅に短縮します。

Scene 04

市場リサーチ

特定の業界や企業について調査したいとき、恭介に依頼するだけで、複数の情報源から横断的にデータを収集し、構造化されたレポートにまとめます。M&A動向、市場規模、競合状況、ターゲット企業のDD用基本情報など、幅広いリサーチに対応します。

リサーチの種類:

  • 業界動向調査 -- 特定業界のM&A件数、主要ディール、市場規模・成長率、バリュエーション水準
  • 企業基本情報 -- 会社概要、沿革、経営陣、財務ハイライト、主要顧客、競合ポジション
  • DD用資料 -- 法的リスク、財務リスク、事業リスク、市場リスクのチェックリスト
  • 競合比較 -- 同業他社との料率比較、サービス比較、市場シェア比較

6セキュリティ・安全性

PE/FA業務では、クライアント情報の守秘義務が極めて重要です。AI Crew は、安全性を最優先に設計されています。

最終判断は必ず人間が行う

AI Crew は、情報収集・分析・ドラフト作成を担当します。しかし、投資判断、クライアント対応、契約に関する最終判断は、必ず上原さんご自身が行う設計になっています。AIが勝手に意思決定することはありません。

たとえば、メールの返信ドラフトを作成しても、実際に送信するかどうかは上原さんの判断です。カレンダーに予定を登録する際も、「登録してよろしいですか？」と必ず確認を取ります。ファイルの編集・削除も、事前確認なしには実行しません。

クライアント情報の守秘義務

Claude Code は、お手元のパソコン上で動作します。データは上原さんのパソコンから直接各サービス（Gmail、Google Calendar等）にアクセスする形で処理され、第三者のサーバーに転送されることはありません。

  • クライアント名や案件情報が外部に漏れることはありません
  • Claude との対話内容は、Anthropic社のサーバーで処理されますが、学習データとしては使用されません（Anthropic社の利用規約に基づく）
  • Google Workspace のデータは、Google の認証基盤を通じて安全にアクセスされます

秘密情報（APIキー等）の保護

各サービスへの接続に使う認証情報（APIキー）は、すべてお手元のパソコン内に保存されます。具体的には：

  • APIキーは .env ファイルとしてローカルに保存
  • .env ファイルは Git（バージョン管理）にアップロードされないよう二重の安全策を設定
  • 万が一、秘密情報が外部に送信されそうになった場合は、自動でブロックする仕組みを導入

ファイル操作の確認フロー

AI Crew には、以下の安全ルールが組み込まれています：

上書き前の確認

既存ファイルを編集する前に、必ず「上書きしますが、よろしいですか？」と確認します。確認なしに内容を変更することはありません。

削除の原則禁止

ファイルの削除コマンドは原則として実行しません。どうしても必要な場合は、対象と理由を明示して承認を得ます。強制削除は一切行いません。

日本語での確認

すべての確認メッセージは日本語で行います。英語の技術用語が必要な場合は、必ず日本語の補足をつけます。

コマンド実行前の説明

技術的なコマンドを実行する際は、「何をするか」「どうなるか」を平易な言葉で説明してから実行します。

重要: PE/FA業務における数値の正確性は極めて重要です。AI Crew は、推測で数値を出すことはしません。データソースが不明確な場合は、その旨を明示し、確認を求めます。

7導入の流れ

セットアップは合計で約1時間半。難しい作業はすべてこちらで行いますので、上原さんはPCの前で見守っていただくだけで大丈夫です。

  1. Claude Code のインストール約10分

    お手元のMacに Claude Code をインストールします。ターミナル（Macに標準搭載されているアプリ）を開いて、1行のコマンドを入力するだけです。こちらで操作しますので、上原さんは見守っていただくだけで結構です。Anthropic社のアカウント作成もこのステップで行います。

  2. MCP接続設定（Gmail、Calendar、Sheets等）約30分

    Gmail、Googleカレンダー、Googleスプレッドシート、Googleドライブ、Slackなど、普段お使いのサービスへの接続を設定します。Google アカウントでの認証（「このアプリにアクセスを許可しますか？」の画面で「許可」を押す操作）が数回必要になります。それ以外はこちらで設定します。

  3. 設定ファイルの配置（CLAUDE.md・キャラ設定等）約10分

    CLAUDE.md、characters.md、各キャラクターの個別設定ファイル、朝チェックスキル、ダッシュボード等のファイルを所定のフォルダに配置します。すべてこちらで行います。

  4. 動作確認・チュートリアル約30分

    実際にClaude Code を起動して、基本的な操作を一緒に試します。「おはよう」と入力して朝チェックが動くか、各キャラクターの呼び出し、メール確認、カレンダー参照などを一通り確認します。上原さんに操作に慣れていただく時間です。

  5. 1週間の試用期間 → フィードバック → カスタマイズ1週間

    セットアップ後、1週間ほど実際の業務で使っていただきます。その間、「ここが使いにくい」「こんな機能がほしい」「キャラの口調を変えたい」などのフィードバックをいただき、カスタマイズを行います。Slackやメールでいつでもご連絡ください。

安心ポイント: 技術的な作業はすべてこちらで行います。上原さんに必要なのは、Googleアカウントの認証（「許可」を押す操作）のみです。セットアップ中に不明点があれば、その場で丁寧にご説明します。

9FAQ（よくあるご質問）

お客様からよくいただくご質問をまとめました。

Q. AIが間違った情報を出したらどうなりますか？
AIは膨大な情報をもとに回答を生成しますが、誤りを含む可能性はゼロではありません。そのため、AI Crew では最終確認は必ず人間（上原さん）が行う設計にしています。特に数値データや企業情報については、AIが情報源を明示するようルール化しており、確認がしやすくなっています。また、「推測で数値を出さない」というルールも組み込まれているため、不確かな情報には「データが見つかりませんでした」と正直に報告します。
Q. データは安全ですか？外部に漏れませんか？
各サービスへの接続に使うAPIキー（認証情報）は、すべて上原さんのパソコン内にのみ保存されます。クラウド上のサーバーに送信されることはありません。Claude との対話内容はAnthropicのサーバーで処理されますが、Anthropic社の利用規約により、お客様の入力データがAIの学習に使用されることはありません。また、Google Workspace のデータは、Google の認証基盤（OAuth）を通じてアクセスするため、Google のセキュリティ基準に準拠しています。
Q. チームメンバーも使えますか？
基本的には1人1セットアップが前提です。各メンバーのClaude Codeに、それぞれのGmailやカレンダーを接続する形になります。ただし、チーム全体で共有するキャラクター設定（世界観やルール）は統一できますので、「チーム全員が同じAI Crewを使っている」という体験を実現できます。チームへの展開をご検討の場合は、別途プランをご提案いたします。
Q. ChatGPT との違いは何ですか？
最大の違いはMCP（Model Context Protocol）によるサービス接続です。ChatGPTは基本的に「対話するだけ」ですが、Claude Code は Gmail、カレンダー、スプレッドシート、Slack等に実際に接続して操作できます。つまり、「調べて」と頼むだけでなく、「メールの下書きを作ってGmailに保存して」「カレンダーに予定を入れて」「スプレッドシートのデータを集計して」といった実際の業務操作までできる点が決定的な違いです。また、CLAUDE.mdによるカスタマイズで、上原さん専用の業務パートナーとして細かく設定できる点もChatGPTにはない特徴です。
Q. プログラミングの知識は必要ですか？
いいえ、一切必要ありません。Claude Code への指示はすべて日本語の自然な言葉で行います。「今日の予定教えて」「メールまとめて」「この業界調べて」のように、普通に話しかけるだけです。初回セットアップも、技術的な作業はすべてこちらで行います。使い方に不安がある場合は、セットアップ時のチュートリアルで丁寧にご説明いたします。
Q. 月額コストはどれくらいかかりますか？
Claude Code の利用には、Anthropic社への月額利用料（月額 $100 〜 $200、約15,000円 〜 30,000円）が発生します。まずは月額 $100 のプランから始めることをお勧めします。通常の業務利用であれば十分な利用枠があります。これに加えて、弊社のサポート費用が別途ございますが、まずは1週間の試用期間で効果を実感していただいた上でご検討いただければと思います。
Q. 後からキャラクターを追加・変更できますか？
はい、いつでも変更可能です。「口調を変えたい」「新しい役割のキャラクターを追加したい」「世界観を変えたい」といったカスタマイズは随時対応いたします。設定ファイル（テキストファイル）を編集するだけなので、対応もスピーディです。
Q. インターネットに接続できない環境でも使えますか？
Claude Code はインターネット接続が必要です。AIの処理自体がクラウド上で行われるため、オフライン環境では利用できません。ただし、一度生成されたレポートやダッシュボード（HTMLファイル）は、ローカルに保存されるため、オフラインでも閲覧可能です。
